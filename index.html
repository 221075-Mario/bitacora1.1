Texto en html

Para el texto en HTML se puede usar de distintas formas como lo que son los encabezados, párrafos, negrita,cursiva , listado ordenado y desordenado, espacios, saltos de líneas y comentarios,
Esto nos sirve para darle una estructura, una mejor legibilidad y accesibilidad a nuestra pagina.

< h3>

Resumen

Es fundamental conocer las distintas etiquetas de HTML, ya que permiten no solo estructurar el texto, sino también darle el estilo, el énfasis y la relevancia adecuada a cada palabra o frase. Al utilizar las etiquetas correctas, puedes plasmar tu contenido de manera clara y precisa, destacando la importancia y el enfoque que cada parte del texto merece, mejorando la experiencia de lectura y la comprensión.

Enlaces en HTML

Ideas principales


Enlaces en HTML

Los enlaces, también conocidos como hipervínculos, son elementos fundamentales en HTML que permiten conectar diferentes recursos dentro de un sitio web o hacia otras páginas externas. Los enlaces no solo sirven para crear vínculos hacia otras páginas o archivos, sino que también son útiles para: Navegación interna: Pueden dirigir al usuario a secciones específicas dentro de la misma página mediante anclas.

Apuntes


Enlaces en HTML

Un enlace es util y se usa colocando una "a" y se usa para facilitar la navegacion.
un enlace dentro de la misma pagina se podria ver asi: < h2 id="contacto">Contacto

Resumen

Los enlaces en HTML, creados con la etiqueta , permiten conectar diferentes páginas, archivos o secciones de una misma página. Utilice el atributo hrefpara especificar la URL de destino. Son esenciales para la navegación interna y externa, la descarga de archivos, y pueden enlazar a correos electrónicos o números de teléfono.

Proceso del diseño

El proceso de diseño, lleva varios puntos a seguir que serian los siguientes:
  1. Fase de análisis del Proyecto.
  2. Información del Usuario.
  3. Benchmarking.
  4. Modelo de Usuario.
  5. diseño del Usuario.
  6. Diseño conceptual.
  7. diseño visual.
Es necesario seguir un proceso estructurado para realizar correctamente el diseño de una página web, ya que de esta manera se logra tener más claridad sobre el objetivo que se desea cumplir con la página ya que tambien se hace un diseño visual para tratar de mostrar como se veria el resultado final recopilando todo el proceso aplicado.



Apuntes


Proceso del diseño

Se utiliza para tener certeza de que el diseño final cumpla la funcion o la necesidad de la pagina web.

Resumen

En resumen, un proceso de diseño metódico es esencial para asegurar que la página web sea efectiva, funcional y esté alineada con las expectativas de los usuarios y los objetivos del negocio.
